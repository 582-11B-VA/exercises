# Exercise: Lists

For this assignment, your task is to implement the functions described
below.

- A function named `surround` that takes a number and returns a list
  containing the number before it, the number itself, and the number
  after it.

- A function named `nums-are-same` that takes a list of three numbers
  and determines whether all three are equal.

- A function named `word-count` that takes a string and returns the
  number of space-separated words it contains (see the built-in function
  [string-split-all]).

[string-split-all]: https://pyret.org/docs/latest/strings.html#%28part._strings_string-split-all%29
