# Exercises: Summarizing

For this assignment, your task is to use `fold` in order to implement
the following functions:

- A function named `avg` that returns the average of a given list of
  numbers.
- A function named `ordered-list` that transform a given list of strings
  into an HTML [ordered list] element.
- A function named `reverse-string` that reverses the characters in a
  given string.

[ordered list]: https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/ol
