# Exercise : Conditionals

For this assignment, your task is to implement a function that
calculates the resell value of a vehicule based on its original price
and how old it is.

- If the vehicle is less than 3 years old, it resells for 80% of the
  original price.
- If it is more than 10 years old, it resells for 50% of the original
  price.
- If the vehicle is at least 3 years old but not older than 10 years, it
  resells for 70% of the original price.
